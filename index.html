Archivo: `script.js` Crea un archivo llamado `script.js`. Este se encargará del scroll suave, de la navegación móvil y del efecto del menú al bajar en la página. ```javascript // Mobile menu toggle const menuToggle = document.querySelector('.menu-toggle'); const navLinks = document.querySelector('.nav-links'); menuToggle.addEventListener('click', () => { navLinks.classList.toggle('active'); }); // Sticky header on scroll const header = document.getElementById('header'); window.addEventListener('scroll', () => { if (window.scrollY > 50) { header.classList.add('scrolled'); } else { header.classList.remove('scrolled'); } }); // Smooth scrolling for navigation links document.querySelectorAll('a[href^="#"]').forEach(anchor => { anchor.addEventListener('click', function (e) { e.preventDefault(); navLinks.classList.remove('active'); // Close mobile menu if open const target = document.querySelector(this.getAttribute('href')); if (target) { window.scrollTo({ top: target.offsetTop - 70, // offset for header height behavior: 'smooth' }); } }); }); // Form submission handler const contactForm = document.getElementById('contactForm'); if(contactForm) { contactForm.addEventListener('submit', function(e) { e.preventDefault(); alert('Gracias por su mensaje. Nos pondremos en contacto a la brevedad.'); contactForm.reset(); }); } ``` Principales características implementadas: - Diseño limpio y moderno: Utiliza la paleta especificada de azul oscuro (`#0a192f`) y azul eléctrico (`#00d2ff`). - Cards y Sombras: Uso de `border-radius` y `box-shadow` suaves, junto con animaciones `transform: translateY(-10px)` al hacer hover. - WhatsApp Integrado: Botón flotante funcional a la derecha y botones dentro de la sección hero y contacto, vinculados al número provisto (`+54 11 6607-0147`). - SEO Ready: Etiquetas `meta description` y `keywords` ajustadas a los servicios solicitados en Buenos Aires y Lanús. - Responsivo: Adaptabilidad para tablets y dispositivos móviles mediante Flexbox y CSS Grid.
